# STREAMED

Streamed is an Android streaming app focused on movies, TV shows, TorBox integration, torrent search, and addon support.

## Features

- Movies and TV shows with TMDB metadata
- VLC-based video playback
- TorBox debrid integration
- Torboxers torrent search
- Stremio addon support
- Magnet management
- DNS-over-HTTPS options

## Development

```bash
pnpm install
pnpm start
```

## Build

```bash
eas build --platform android --profile preview
```
